Browser cache:

Browser cache plays a vital role in web performance by decreasing the loading time. When a user visit any website and interct resources on the website it cache information for some period of time with expiry date. If user visit same website or visit different page on same site for second time then the browser cache shows result of resources without requesting resources to the server. By doing this it improve the faster loading time and user friendly browsing experience.

In the fast paced world user does not wait for resources to be open with dalays. Study done by GOOGLE that 53% user abondoned resources if it takes longer then 3 second to load resources. By implementing the browser cache web owner lower the load time and decrase the abondoned rate, improve the user satisfaction.

How it works?

Browser caching work by instructing the user browser for storing the resourceslocal by specific time being. When any user visit the resources a rseponse header send to collect the information for time being. This browser caching is done by using the HTTP header such as cache-control which spcifies the caching policies. So when same user visit again same resources then data fetch by cache instead of requesting to server.
