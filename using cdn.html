Using CDN for enhancing web performance:

Content Delivery Network have emerged for delivering the fast paced very quick content over the world wide. Content delivery network simply means that placing the server in stratigacally location aimed to enhanced the delivery of the content over different location of the users.When a user placed the request for the content then CDN play an important role in deleviring those content to the user who requested. It direct the request to the nearst server to minimise the time and the data travel distance.

Benefits of CDN:

CDN solely designed to optomise the delivery of the content.CDN provide many advantages to its user included individuals and companies.
It has following main benefits:

  • Faster load time


  • Enhanced website security


  • Improved availability


  • Reduced Bandwidth costs
