Lazy loading:

Lazy loading is one of the the well known and widely used technique for enhancing the web performance. In lazy loading only certain resources like images and videos loaded as needed. It load other resources later when the user interact with those resources by scrolling down. This way only upfront page with important data loaded and remain unintrected data remain unloaded.

Benifits of lazy loading:

Lazy loading technique play vital role in the website performance and boost the website efficiency. It has following benefits:

  • Lazy loading improve the initial page loading by loading ony viewable resources which also reduce the website latency.

  • Lazy loading prevent the downloading of the unnecessary data by loading the interected resources which also reduce the bandwidth. On mobile phone internet speed varies so loading first page faster lower the strain on bandwidth by showing the intrected media only.

  • Faster page load means better rank on the search engine beneficial for both ownner and make website friendly for user.


  • Overall delaying the unintrected resources typically enhanced the web performance.
